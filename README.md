# matsu.num.Statistics.KernelDensity
`matsu.num.Statistics.KernelDensity` は Java 言語向けの数値計算における, 
カーネル密度推定を扱うライブラリである.

現在はリリースバージョンは `0.4.0` (開発中バージョン) であり, Java 17 に準拠する.
このバージョンにおいて, 次がサポートされている.

- 1次元のカーネル密度推定
    - ガウスカーネルによる1次元カーネル密度推定
- 2次元のカーネル密度推定
    - ガウスカーネルによる2次元カーネル密度推定
